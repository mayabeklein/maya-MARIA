## 🏗️ שלב 2: יצירת דף חדש (New Page)

### 2.1 הוספת דף חדש

1. בבונה האפליקציה, תראי רשימת דפים קיימים
2. לחצי על כפתור **"+ Add Page"** או **"Create New Page"**
3. תיפתח חלונית עם אפשרויות

### 2.2 בחירת סוג הדף

בחרי ב: **"Landing Page"** או **"Product Page"**

**למה?** כי זה הסוג המתאים ביותר למכירת מוצר.

### 2.3 הגדרות בסיסיות

**שם הדף (Page Name):**
```
Balance Now - גברים
```

**כתובת URL (Slug):**
```
balance-now-men
```
או
```
gavrim-60
```

**תיאור (Description):**
```
תוכנית ליווי גברית ל-60 יום - להוריד בטן, להחזיר אנרגיה, לאזן הורמונים
```

## 🎨 שלב 3: עיצוב הדף (Page Design)

### 3.1 בחירת תבנית (Template)

Kajabi מציעה כמה תבניות מוכנות:

- **"Hero + CTA"** - מומלץ! (תמונה גדולה + כפתור)
- **"Product Showcase"** - טוב למוצרים
- **"Simple Landing"** - מינימליסטי

**המלצה:** בחרי ב-**"Hero + CTA"**

### 3.2 הוספת תמונת רקע (Hero Image)

1. לחצי על אזור התמונה הראשית
2. לחצי **"Upload Image"** או **"Choose from Library"**
3. העלי תמונה של:
   - גבר בריא ואנרגטי (40-50)
   - רקע נקי ומקצועי
   - לא תמונת חדר כושר!

**גודל מומלץ:** 1200x800 פיקסלים

**איפה למצוא תמונות?**
- ב-Canva שלך (אם יש)
- ב-Google Drive (תיקיית תמונות)
- או השתמשי בתמונה זמנית ותחליפי מאוחר יותר

## ✍️ שלב 4: כתיבת התוכן (Content)

### 4.1 כותרת ראשית (Headline)

**העתיקי את זה בדיוק:**

```
תוכנית ליווי גברית, חדה וממוקדת ל-60 יום
```

**הגדרות עיצוב:**
- גופן: **Bold** (מודגש)
- גודל: **32-40px**
- צבע: **שחור** או **כחול כהה**
- יישור: **מרכז**

### 4.2 תת-כותרת (Subheadline)

**העתיקי את זה:**

```
בנוי לגברים עסוקים שרוצים להוריד את הבטן,
להחזיר אנרגיה ולבנות שליטה אמיתית -
בלי לוותר על החיים הטובים.
```

**הגדרות עיצוב:**
- גופן: **Regular** (רגיל)
- גודל: **18-22px**
- צבע: **אפור כהה** (#333333)
- יישור: **מרכז**

### 4.3 נקודות מפתח (Key Points)

**הוסיפי רשימה של 5 נקודות:**

```
✅ פרוטוקול תזונה גברי ברור
✅ שבוע RESET הורמונלי
✅ מערכת הרגלים שאפשר לחיות איתה
✅ ליווי באפליקציה בעברית
✅ קבוצה גברית אמיתית
```

**הגדרות עיצוב:**
- גופן: **Medium**
- גודל: **16-18px**
- צבע: **שחור**
- רווח בין שורות: **1.5**

## 🔘 שלב 5: הוספת כפתור קריאה לפעולה (CTA Button)

### 5.1 יצירת הכפתור

1. לחצי **"Add Element"** או **"Add Button"**
2. גררי את הכפתור למקום הנכון (מתחת לתוכן)

### 5.2 טקסט הכפתור

**בחרי אחד מאלה:**

```
אני רוצה להתחיל עכשיו
```
או
```
הצטרף עכשיו
```
או
```
קנה עכשיו - ₪1,790
```

**המלצה:** השתמשי ב-**"אני רוצה להתחיל עכשיו"** - זה יותר מעורר פעולה.

### 5.3 עיצוב הכפתור

**צבע רקע:** 
- **כחול חזק** (#0066CC)
- או **ירוק** (#28A745)
- או **כתום** (#FF6B35)

**המלצה:** כחול או ירוק - גברי יותר.

**טקסט:**
- צבע: **לבן** (#FFFFFF)
- גופן: **Bold**
- גודל: **18-20px**

**גודל הכפתור:**
- רוחב: **מלא** (Full Width) או **300px**
- גובה: **50-60px**

**פינות:** מעוגלות קלות (8-12px)

### 5.4 קישור הכפתור (Button Action)

**לאן הכפתור יוביל?**

**אופציה 1: לדף תשלום (מומלץ!)**
```
Link to: Checkout Page
Product: Balance Now - גברים
```

**אופציה 2: לדף מכירה מלא**
```
Link to: Sales Page
Page: balance-now-men-full
```

**אופציה 3: לטופס הרשמה**
```
Link to: Form
Form: Balance Now Registration
```

**המלצה:** אופציה 1 - ישירות לתשלום!

## 📝 שלב 6: הוספת אלמנטים נוספים (Optional)

### 6.1 סרטון הסבר (Video)

אם יש לך סרטון קצר (1-2 דקות):

1. לחצי **"Add Element" > "Video"**
2. העלי סרטון מ-Vimeo או YouTube
3. שימי אותו **מתחת לכותרת** או **מעל הכפתור**

### 6.2 המלצות (Testimonials)

אם יש לך המלצות מגברים:

1. לחצי **"Add Element" > "Testimonial"**
2. הוסיפי:
   - ציטוט
   - שם + גיל
   - תמונה (אופציונלי)

**דוגמה:**
```
"ירדתי 8 קילו ב-60 יום, אבל הדבר האמיתי זה האנרגיה.
אני חד, מרוכז, ומרגיש צעיר ב-10 שנים."

- יוסי כהן, 45
```

### 6.3 שאלות נפוצות (FAQ)

1. לחצי **"Add Element" > "Accordion"** או **"FAQ"**
2. הוסיפי 3-5 שאלות נפוצות:

**דוגמה:**
```
❓ אין לי זמן לבשל, זה מתאים לי?
✅ בהחלט. התוכנית בנויה לגברים עסוקים.

❓ מה אם אני לא מתאמן?
✅ התוכנית עובדת גם בלי אימונים.
```

## 💾 שלב 7: שמירה ופרסום

### 7.1 שמירה

1. לחצי **"Save"** או **"Save Draft"** (למעלה מימין)
2. הדף נשמר כטיוטה

### 7.2 תצוגה מקדימה (Preview)

1. לחצי **"Preview"** (תצוגה מקדימה)
2. בדקי איך הדף נראה באפליקציה
3. בדקי שהכפתור עובד

### 7.3 פרסום (Publish)

**רק אחרי שבדקת והכל טוב:**

1. לחצי **"Publish"** (פרסום)
2. הדף יהיה זמין באפליקציה!

---

## 📲 שלב 8: הוספת הדף לתפריט האפליקציה

### 8.1 ניווט לתפריט

1. בבונה האפליקציה, לחצי על **"Navigation"** או **"Menu"**
2. תראי את התפריט הראשי של האפליקציה

### 8.2 הוספת פריט תפריט חדש

1. לחצי **"+ Add Menu Item"**
2. בחרי **"Page"**
3. בחרי את הדף שיצרת: **"Balance Now - גברים"**

### 8.3 הגדרות התפריט

**שם בתפריט:**
```
Balance Now לגברים
```
או
```
תוכנית 60 יום
```

**אייקון:** בחרי אייקון מתאים (💪 או 🔥 או ⚡)

**מיקום:** שימי אותו **למעלה** בתפריט (גרירה)
